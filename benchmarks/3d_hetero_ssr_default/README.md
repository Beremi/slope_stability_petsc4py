# 3d_hetero_ssr_default

Runnable case configuration.

Files:
- `case.toml`
- `run.sh`
- `simulation.ipynb`
- `visualisation.ipynb`

Run with:

```bash
./run.sh
```

By default outputs go to `artifacts/cases/3d_hetero_ssr_default/latest`.

Notebook workflow:

- `simulation.ipynb` forces a fresh solver run by default and writes notebook-local artifacts under `artifacts/simulation/`.
- `visualisation.ipynb` reuses those artifacts and rebuilds the MATLAB-parity plots.
- Shared notebook helpers live in `benchmarks/notebook_support.py`, and all benchmark notebooks are generated by `benchmarks/generate_benchmark_notebooks.py`.

Historical analysis archive:

- Legacy scaling, comparison, and solver-investigation scripts and reports live under `archive/`.
- The benchmark root now keeps only the canonical runnable files listed above.
